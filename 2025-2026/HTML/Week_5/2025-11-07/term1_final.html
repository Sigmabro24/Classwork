Phase Description
Idea phase This is where they develop the plan and make the documentation for the project
Planning Phase This is where they figure out what to do in chronological order with end goals in mind
Analysis Phase They create requirements for the project and overall analyze it
Coding Phase This is where they implement their previous work and write it down as code and test it multiple times to get rid of bugs
Deployment Phase DevOp make sure that they can bring down the code and allow end users to access it freely
Maintenance Phase This is where they fix up the code and make sure its working
