Week 3 Vocabulary

Peripherals Extra devices that are not needed to make the computer work but are helpful using it.
Integer Whole numbers that are positive and negative
Structural Form The visual appearance of code
Separation of Concerns Something that allows us to organize our code into multiple sections each containing their own function
Debugging Fixing your code and making it functional.
Float Numbers with fractions or decimal points
Branching Something that changes the result by altering the value but leaving the element untouched
CPU The part of the computer that allows the computer to work and process information
Bool A piece of code with true or false values.
Variable Something that stores information for later use.
Editor Shortcut Key combinations that allow for quicker and more simple navigation.
None the "Null" A piece of code that holds things without value and holds it till the value changes.
